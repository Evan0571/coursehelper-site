A focused workspace for the class loop.

CourseHelper is shaped around what happens before class, during class, and after class, instead of treating every note as a blank page.

Course organization

Keep courses, sessions, assignments, exams, materials, and calendar sync in one place.

Recording to summary

Record or import class audio, then generate transcripts and summaries for review.

Review artifacts

Create flashcards, quizzes, and key terms from class context when it is time to study.

From syllabus to review.

The product is designed for repeated weekly use: set up the class, capture what happened, turn it into study material, and come back when exams are close.

Use flashcards, quizzes, key terms, and AI answers when you return to the course.

Built with clear recording and privacy boundaries.

CourseHelper is a study assistant. Students are responsible for getting any required permission before recording or uploading class content, and AI output should be checked against course materials.
